PINOUT FOR ESP32
================

This library uses VSPI bus ESP32



MOSI   23 (GPIO23)   -->      MOSI

MISO   19 (GPIO19)   -->      MISO

CLK    18 (GPIO18)   -->      CLK

CS      5 (GPIO5)    -->      CS

PIN PD HOTMCU

One resistor of 47K to 3,3v

One condensator of 100k (o,1uF) to GND
